// Why LANGUAGE

Built Different.

LANGUAGE is an experiment in building a programming language from scratch, one feature per day. Clean syntax. PascalCase keywords. Python-like indentation. No boilerplate.

[01]

Minimal Syntax

No semicolons, no curly braces. Indentation-based blocks with PascalCase keywords that read like plain English.

[02]

Cross Platform

Runs on Windows, Linux, and macOS. Available as an RPM for Fedora or a native installer for Windows.

[03]

Zero Dependencies

Compiled to a single standalone binary. No runtime, no VM, no DLLs. Just download and run.

[04]

VS Code Extension

Full syntax highlighting available on the VS Code Marketplace. Install in one click.

[05]

Open Source

MIT licensed. Built in C++17. Every line of the interpreter is open for you to read, fork, and contribute to.

[06]

Growing Fast

New features every day. Networking, UI, compilation, and even a game engine are all on the roadmap.

// See It In Action

Clean. Readable. Powerful.

LANGUAGE reads almost like pseudocode. If you can think it, you can write it. No syntax getting in the way of your ideas.

fizzbuzz.LANGUAGE 
# FizzBuzz in LANGUAGE #
For i = 1 To 20
  If i == 15
    Print "FizzBuzz"
  Elif i == 3
    Print "Fizz"
  Elif i == 5
    Print "Buzz"
  Else
    Print i
  End
End

# Functions & arrays #
Func sum(nums)
  total = 0
  i = 0
  While i < Length(nums)
    total = total + nums[i]
    i = i + 1
  End
  Return total
End

scores = [10, 25, 37, 42]
Print sum(scores)

// Get It

Install LANGUAGE

Available on Linux and Windows. Download the latest release from GitHub or install via package manager on Fedora.

Fedora / RHEL

$ sudo dnf install ./LANGUAGE-0.5.0-Linux-x86_x64.rpm

Windows

> LANGUAGE-0.5.0-Setup.exe

Build from Source

$ git clone https://github.com/B16SETC/LANGUAGE-Programming-Language
$ cd LANGUAGE && mkdir build && cd build
$ cmake .. && cmake --build .
